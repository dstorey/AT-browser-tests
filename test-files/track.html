video + track element test

Pass Criteria

  • #1: Elements are supported
  • #2: Elements have correct HTML and Core Accessibility API mappings
  • #3: At least one subtitle format is supported (WebVTT or TTML)
  • #4: Controls are keyboard accessible:
    • #4.1: Controls MUST be focusable
    • #4.2: MUST be able to activate controls, including any child menu/controls

Reference

  • HTML5 video element
  • HTML Accessibility API Mappings for video element
  • HTML5 track element
  • HTML Accessibility API Mappings for track element

Test vd-001: Video & track elements with WebVTT subtitles

Code

<video id="video1" controls width="300" height="300">
	<source src="../video/ElephantsDream.mp4" type="video/mp4">
	<source src="../video/ElephantsDream.webm" type="video/webm">
	<track kind="captions" srclang="en" src="../video/subtitles-en.vtt" label="English">
</video>
